## 2. hook_ssl2.js

```text
//hook_ssl.js不好用了后，再使用hook_ssl2.js

2. adb push certs.dex /data/local/tmp/certs.dex
3. 执行 hook_ssl2.js，收集chain
frida -U --no-pause -f xxxxxxxx.package.name -l hook_ssl2.js
4. 打开抓包
```
